## üé• PRODUCTION TIPS

### Visual Style
- Use screen recording at 1920x1080 or 4K
- Show cursor movements clearly
- Use zoom-in for small UI elements
- Add subtle transitions between sections

### Pacing
- Speak slowly and clearly
- Pause 1-2 seconds between major actions
- Allow UI animations to complete before moving on

### Annotations
- Add text callouts for key information
- Highlight buttons before clicking
- Use arrows to point out important features

### Audio
- Use a quality microphone
- Record in a quiet environment
- Add subtle background music (non-distracting)

### Editing
- Cut long loading/waiting times
- Speed up repetitive actions (form filling)
- Add chapter markers for easy navigation
